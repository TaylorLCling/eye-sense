<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* OpenCV
* CMake
* libgtk2.0-dev
* pkg-config


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/taylorlcling/EyeSense.git
   ```
2. Generate Build Files using cmake
   ```sh
   cmake .
   ```
3. Build Program
   ```sh
   make
   ```
4. Run Resulting Executable
   ```sh
   ./EyeSense
   ```
5. Press any key to Exit
